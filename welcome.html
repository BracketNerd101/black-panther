Welcome Survey

Problem

Re-design & update an introduction survey for new insurance members. This application’s main function is to gather high-level information about new insurance members.

Solution

Create a simple and dynamic interface that displays survey questions in a card format. The main concept of the redesign was based on gamification. This gamification concept helps the user feel empowered an involved in completing the survey.
